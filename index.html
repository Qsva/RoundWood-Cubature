Калькулятор кубатуры круглого леса

В графе длина - указать длину бревна для которого, ведется расчет. Только один размер за раз

В графе диаметр - через запятую, без пробелов и других символов, указывать диаметр бревна в см.

Важно! Разделить между цифрами запятая, без лишних символ, в конце должна быть только цифра, значения только двузначные. Например бревно 200мм. - пишем 20. Калькулятор написан опираясь на данные ГОСТ таблицы для кубатуры бревен, поддерживает длины от 1м. - 8,5м.!
